❓ FAQ

Quick answers

What is StudyMate AI? ▼
StudyMate AI is a full-featured, offline-first Android study companion app built with Flutter. It combines AI-powered learning tools, gamification, social features, and smart scheduling into one seamless experience. Features include FSRS spaced repetition flashcards, Gemini AI mentor, XP-based leveling, 23 achievements, QR friend system, and detailed analytics.
What tech was used to build it? ▼
Built with Flutter 3.44.4 and Dart 3.12.2. Primary database: Firebase Firestore for cloud sync + real-time features. Local storage: SQLite (sqflite 2.4.3) with SharedPreferences dual-write for offline-first. State management via Riverpod 2.6.1. AI powered by Google Gemini API. Charts with fl_chart 0.69.2. QR scanning via mobile_scanner. Chat via Firestore real-time listeners. XP Shop with 10 items. Education System with 108+ subjects across Science/Commerce/Arts streams. Build system: Gradle 8.11.1 / AGP 8.9.1 with JDK 21.
How does the gamification system work? ▼
There are 13 XP event types: study sessions (10 XP/min), quizzes (50 XP), perfect scores (100 XP), flashcard reviews (5 XP), notes (15 XP), adding friends (50 XP), daily login (10 XP), and more. Level formula: N² × 100 XP. There are 23 auto-checked achievements, daily login streaks, 4 leaderboard types, and a daily reward system based on streak length.
What is FSRS spaced repetition? ▼
FSRS (Free Spaced Repetition Scheduler) is a next-generation algorithm for flashcard reviews. It uses 4 rating levels — Again, Hard, Good, Easy — and tracks three metrics per card: stability (how long you'll remember), difficulty (how hard the card is), and retrievability (probability of recall today). This optimizes review timing for maximum retention with minimum effort.
Can I use it offline? ▼
Yes! StudyMate AI is built offline-first. All core features — flashcards, notes, quizzes, focus timer, analytics, achievements — work completely without internet. The only feature requiring internet is the AI Mentor (Gemini API). Your data is stored locally on your device using SQLite, ensuring privacy and availability anywhere.
